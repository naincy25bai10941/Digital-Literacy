# Digital-Literacy
This project helped me understand digital tools, online safety, and how to use platforms like GitHub, LinkedIn, and Google Forms.
## 📌 About the Project
This project is based on the concept of digital literacy and how students can use digital tools in a better and more responsible way. While working on this, I explored different platforms, created some basic content, and also learned about online safety and communication.

At first, I was not very familiar with some of the tools, but after trying them step by step, it became easier to understand. Overall, this project helped me gain some practical knowledge along with theory.
## Task Included
### Task 1 - Infographic
I created a simple infographic using Canva covering topics like digital literacy, safe internet use, and useful tools for students.
### Task 2 - Digital Portfolio
I created and updated my profiles on:
* GitHub  
* LinkedIn  
* Kaggle
  
These platforms helped me understand how to maintain a basic professional presence online.
